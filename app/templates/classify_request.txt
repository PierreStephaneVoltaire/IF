| Tier | Description | Use Case |
|------|-------------|----------|
| tier1 | Simple, fast | Casual chat, greetings, proofreading |
| tier2 | Standard technical | Basic coding, explanations, shell commands |
| tier3 | Complex reasoning | Multi-file changes, code reviews, detailed analysis |
| tier4 | Highest quality | Architecture decisions, critical analysis |

## Rules

- Technical/coding flows always start at tier2
- Nuanced/complex analysis starts at tier3
- General conversation can range tier1-tier4 depending on complexity
- **Set websearch=true** if question involves:
  - Current events, recent news
  - Dates, time-sensitive information
  - Financial data, stock prices
  - Real-time info (weather, sports scores)
